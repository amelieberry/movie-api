Movie API Documentation

Objective

The web application provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Features

  • Return a list of all movies
  • Return (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

API Endpoints

{ "Name": "Rob Minkoff", "Bio": "Robert Ralph Minkoff is an American filmmaker. He is known for directing the double Academy Award-winning animated feature The Lion King, along with directing Stuart Little.", "Birth": "1962" }
Get a list of all users /users GET - - A JSON object holding data about all the users
Allow new users to register /users POST - A JSON object holding data about the user to add, structured like:
{ "Username": "Anonymous", "Password": "greatPassword3", "Email": "email3@gmail.com", "Birthday": "03/01/1995", } 		A JSON object holding data about the new user, including their created ID, structured like:
{ "Username": "Anonymous", "Password": "greatPassword3", "Email": "email3@gmail.com", "Birthday": "1995-02-28T23:00:00.000Z", "FavoriteMovies": [], "_id": "634ed27075eda577717eaea4", "__v": 0 }
Allow users to update their user info /users/:Username PUT :Username (the Username of the user) A JSON object holding data about the existing user, structured like: { "Username": "Anonymous", "Password": "UPDATEDgreatPassword3", "Email": "email3@gmail.com", "Birthday": "03/01/1995", } A JSON object holding the updated user data, including their ID, structured like:
{ "_id": "634ed27075eda577717eaea4", "Username": "Anonymous", "Password": "UPDATEDgreatPassword3", "Email": "email3@gmail.com", "Birthday": "1995-02-28T23:00:00.000Z", "FavoriteMovies": [], "__v": 0 }
Allow users to add a movie to their list of favorites /users/:Username/:movies/:MovieID POST :Username (the Username of the user), :MovieID (the ID of the added movie) - A JSON object holding the updated user data, including their ID, structured like:
{ "_id": "634ed27075eda577717eaea4", "Username": "Anonymous", "Password": "UPDATEDgreatPassword3", "Email": "email3@gmail.com", "Birthday": "1995-02-28T23:00:00.000Z", "FavoriteMovies": [ "6349736b3277271a18972f8c" ], "__v": 0 }
Allow users to remove a movie from their list of favorites /users/:Username/:movies/:MovieID DELETE :Username (the Username of the user), :MovieID (the ID of the movie to be removed) - A text message indicating whether the movie was successfully removed from favorites
Allow existing users to deregister /users/:Username DELETE :Username (the Username of the user) - A text message indicating whether the user was successfully deregistered
