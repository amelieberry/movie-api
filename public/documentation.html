Movie API Documentation

Objective

The web application provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Features

  • Return a list of all movies
  • Return (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description and movies) by name (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year. and movies) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Parameters

-

URL

/users

HTTP Method

POST

Parameters

-

Request body data format

A JSON object holding data about the user to add, structured like: 


{ 
    "Username": "Anonymous", 
    "Password": "greatPassword3", 
    "Email": "email3@gmail.com", 
    "Birthday": "03/01/1995", 
}

Response body data format

A JSON object holding data about the new user, including their created ID, structured like: 


{ 
    "Username": "Anonymous", 
    "Password": "greatPassword3", 
    "Email": "email3@gmail.com", 
    "Birthday": "1995-02-28T23:00:00.000Z",
    "FavoriteMovies": [], 
    "_id": "634ed27075eda577717eaea4", 
    "__v": 0 
}

URL

/users/:Username

HTTP Method

PUT

Parameters

:Username (the Username of the user)

Request body data format

A JSON object holding data about the existing user, structured like: 


{ 
    "Username": "Anonymous", 
    "Password": "UPDATEDgreatPassword3",
    "Email": "email3@gmail.com", 
    "Birthday": "03/01/1995", 
}

Response body data format

A JSON object holding the updated user data, including their ID, structured like: 


{ 
    "_id": "634ed27075eda577717eaea4", 
    "Username": "Anonymous",
    "Password": "UPDATEDgreatPassword3", 
    "Email": "email3@gmail.com",
    "Birthday": "1995-02-28T23:00:00.000Z", 
    "FavoriteMovies": [], 
    "__v": 0 
}
